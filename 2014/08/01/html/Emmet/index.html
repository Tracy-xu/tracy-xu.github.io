Emmet

2014-08-01 Web

在前端开发的过程中，一大部分的工作是写 HTML、CSS 代码。特别是手动编写 HTML 代码的时候，效率会特别低下，于是，就有了 Emmet，它可以极大的提高代码编写的效率。它提供了一种非常简练的语法规则，然后立刻生成对应的 HTML 结构或者 CSS 代码，同时还有多种实用的功能帮助进行前端开发。

HTML 语法

  • 生成 HTML 文档初始结构
  • 生成带有 id 、class 的标签
  • 生成后代标签 >
  • 生成兄弟标签 +
  • 生成上级标签 ^
  • 重复生成多份标签 *
  • 生成分组 ()
  • 生成自定义属性 [attr]
  • 对生成内容编号 $
  • 生成文本内容 {}

注：要注意前后的符号关系。
