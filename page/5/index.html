CSS 单位

2013-06-22 Web

PX

px 是一个绝对单位，不相对于谁，只受平台 dpi 影响。cm、pt 之类的也都是绝对长度，但与 px 这个抽象单位不同的是它们都是物理单位，1pt 则是 1/72 英寸，而 1 英寸换算到公制是 2.54cm，在 CSS 中所有的物理单位会直接映射到像素，在 96dpi 下 1in = 2.54cm = 96px 1cm = 37.8px。
