WSL 安装

  • 命令行安装
  • 微软商店安装

在“启动或关闭 Windows 功能”中勾选“适用于 Linux 的 Windows 子系统”及“虚拟机平台”，点确定，重新启动系统。

打开 “Windows Store” 搜索 wsl，点击安装 “Ubuntu”

Ubuntu 安装过程中会让设置用户名和密码，该账户默认为该 Linux 系统的管理员，具备和 root 相同的权限（root 已经有了不能起重复的账户名）。

注意：Ubuntu 默认不允许使用 root 登录（报 su: Authentication failure），初始 root 账户不能使用，需要在普通账户下利用 sudo 权限 sudo passwd root 设置 root 密码，再 su root 切换到 root。

  • 与 VSCode 一起使用

VSCode 中可通过 “Remote - WSL” 插件连接 WSL 子系统。

安装完成后，会出现“远程资源管理器”图标，点击进去后，点击 “Connect to WSL”，即可用 VSCode 链接上 WSL。

“CTRL + K + O” 输入 /mnt/，该目录挂载了 Windows 中的文件系统，通过此处可在 WSL 中访问 Windows 文件系统。

  • Windows 访问 WSL

资源管理器右侧可以进入 WSL。
