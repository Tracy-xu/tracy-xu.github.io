2016-02-06 css

postcss 是一个 CSS 编译器，可用来…。

github postcss 项目上的描述：

postcss is a tool for transforming styles with JS plugins. These plugins can lint your CSS, support variables and mixins, transpile future CSS syntax, inline images, and more.

postcss 通过插件，来对 CSS 处理一些前置或后置的事情。

注：postcss 不是预处理器（Preprocessor），也不是后处理器（Postprocessor），它集成了前置和后置的处理功能，可称之为”处理器”。”post”应理解为”加工”而不是”Postprocessor”。

特点

插件

像postcss 本身一样令人惊奇的是，它有很多不同的插件，这也是让postcss 发光之处。当你看到postcss 在GitHub上的可用插件列表，你会发现其具有各式各样的功能。

postcss 是用 JavaScript 编写的插件

使用

postcss 的使用方式有很多种，可以单独使用，也可以结合其他自动化工具：npm run、grunt、gulp、webpack、浏览器。

插件

另一方面，postcss 具有很可用的插件，你可以根据自己所需尽可能使用多的选择或尽可能少选择。
