科里化函数

科里化的关键是返回函数实现连式调用、参数收集、以及判断收集结束做最终调用。判断收集结束有很多办法：对比实参和形参的长度；判断实参的个数是否为 0（不传）。

上面版本中，对被科里化的函数没要求，但需要手动结束调用。

这个版本中，不需手动结束调用，但被科里化的函数的形参必须要有长度。
